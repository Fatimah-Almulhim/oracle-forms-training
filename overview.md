# Simple CRUD Form (Oracle Forms)

## What is this project?
This project explains a simple CRUD form using Oracle Forms.
It is designed for learning and training purposes.

## What does CRUD mean?
- Create: Add new record
- Read: View records
- Update: Edit existing record
- Delete: Remove record

## Why this project?
- To understand Oracle Forms basics
- To practice buttons and triggers
- To apply concepts in a simple way
